✊✋✌ Joken Pô – Pedra, Papel e Tesoura

Um jogo moderno de Pedra, Papel e Tesoura desenvolvido com HTML, CSS e JavaScript puro, com visual neon, efeito glassmorphism e vídeo animado de fundo.

Projeto focado em manipulação de DOM, eventos, lógica de jogo e carregamento dinâmico de imagens.

🎮 Demonstração

O jogo permite que o jogador escolha entre:

🪨 Pedra

📄 Papel

✂️ Tesoura

A máquina faz uma escolha aleatória e o sistema:

Determina o vencedor

Atualiza o placar

Exibe as imagens escolhidas

Destaca o vencedor da rodada

✨ Funcionalidades

✔ Interface moderna com efeito Neon Glow
✔ Vídeo de fundo responsivo
✔ Sistema de pontuação dinâmica
✔ Destaque visual do vencedor
✔ Carregamento dinâmico de imagens via JSON
✔ Fallback automático para imagens locais ou online
✔ Totalmente responsivo (mobile, tablet e desktop)
✔ Código organizado e comentado

🛠 Tecnologias Utilizadas

HTML5
CSS3 (Flexbox, Media Queries, Glassmorphism, Animações)
JavaScript Vanilla (ES6+)
